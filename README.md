🤖 EV3 Autonomous Robot Navigation

This project is a MicroPython-based autonomous navigation system for the LEGO EV3 robot. The robot is designed to explore unknown environments, detect and avoid obstacles, and navigate using a combination of reactive and pathfinding algorithms.

🧠 Algorithms Used
A* Pathfinding Algorithm for optimal route planning
Random Mouse Algorithm for environment exploration
Right-Hand Rule (Wall Following) for maze navigation
⚙️ Features
Autonomous movement and decision making
Obstacle detection and avoidance using sensors
Hybrid navigation system (exploration + path optimization)
Return-to-start capability
Modular and readable EV3 MicroPython code
🛠️ Hardware
LEGO Mindstorms EV3 Brick
Ultrasonic Sensor
Color Sensor
Gyro Sensor
EV3 Motors
📌 Purpose

Developed as a robotics project to demonstrate autonomous navigation, real-world algorithm implementation, and sensor-based decision making on embedded systems.
